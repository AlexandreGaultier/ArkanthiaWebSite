# 🎮 Arkanthia - Site Web Communauté Gaming

## 🌟 Description

Arkanthia est un site web moderne et élégant pour une association organisant des bals à thème féerique. Le site présente une ambiance mystique et sombre, inspirée des univers fantastiques, avec une typographie distinctive utilisant la police Starstruck pour les titres.

## ✨ Fonctionnalités

### 🎨 Design & Ambiance
- **Thème féerique sombre** avec des couleurs mystiques
- **Police Starstruck** pour les titres principaux
- **Animations fluides** et effets visuels enchanteurs
- **Design responsive** adapté à tous les appareils
- **Effets de parallaxe** et animations au scroll

### 📱 Sections Principales
1. **Landing Page** - Présentation de l'association
2. **Qui Nous Sommes** - Histoire et statistiques
3. **Prochain Bal** - Mise en avant de l'événement à venir
4. **Lore** - Univers des royaumes mystiques
5. **FAQ** - Questions fréquentes
6. **Contact** - Informations de contact

### 🎭 Contenu Féerique
- **Royaume de Vordemir** - Montagnes enneigées et cités cachées
- **Royaume d'Al Bagarde** - Gardiens de connaissances anciennes
- **Bals thématiques** avec costumes et danses traditionnelles
- **Expériences immersives** pour les participants

## 🛠️ Technologies Utilisées

- **HTML5** - Structure sémantique
- **CSS3** - Styles modernes avec variables CSS
- **JavaScript** - Animations et interactions
- **Google Fonts** - Typographie complémentaire
- **Unsplash** - Images de haute qualité

## 🚀 Installation et Utilisation

### Prérequis
- Navigateur web moderne
- Serveur web local (optionnel)

### Installation
1. Clonez le repository
2. Ouvrez `index.html` dans votre navigateur
3. Ou servez les fichiers via un serveur web local

### Développement
```bash
# Pour un serveur local simple (Python)
python -m http.server 8000

# Ou avec Node.js
npx serve .
```

## 🔧 Personnalisation

### Modifier les Couleurs
Éditez les variables CSS dans `css/styles.css` :
```css
:root {
    --primary-color: #1a1a2e;
    --secondary-color: #16213e;
    --gold-color: #e94560;
    /* ... */
}
```

### Ajouter du Contenu
- **Nouveaux événements** : Modifiez `evenements.html`
- **Lore supplémentaire** : Ajoutez dans `lore.html`
- **FAQ** : Étendez `faq.html`

### Modifier les Images
Remplacez les URLs Unsplash dans le CSS par vos propres images :
```css
background-image: url('votre-image.jpg');
```
